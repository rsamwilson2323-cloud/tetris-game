# 🧱 Falling Blocks (Tetris Game)

A classic Tetris-style Falling Blocks game built using HTML, CSS, and JavaScript with canvas rendering. Features smooth gameplay, block rotation, collision detection, line clearing, and score tracking with responsive keyboard controls.

---

## 🚀 Features

- 🧱 Classic Tetris gameplay  
- 🎮 Smooth falling block mechanics  
- 🔄 Block rotation system  
- ⚠️ Collision detection  
- 🧹 Line clearing system  
- 📊 Score tracking  
- ⌨️ Keyboard controls  
- ⚡ Fast and lightweight (no external libraries)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Canvas API)

## 🎯 Controls

- ⬅️ Arrow Left → Move left  
- ➡️ Arrow Right → Move right  

---

## 🎯 Gameplay

- Blocks fall from the top  
- Arrange them to complete full horizontal lines  
- Completed lines are cleared  
- Score increases for each cleared line  
- Game ends when blocks reach the top :contentReference[oaicite:0]{index=0}
